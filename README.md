# Wi-Fi Password Extractor (Windows)

This Python script retrieves all saved Wi-Fi profiles and their stored passwords on a Windows system using the `netsh` command.

⚠️ **Important:**  
This tool is for **educational and personal use only**.  
Do not run this on any computer without permission — it is illegal.

---

## Features
- Lists all saved Wi-Fi network names (SSIDs)
- Extracts stored Wi-Fi passwords (Key Content)
- Works on all Windows versions that support `netsh`

---

## Requirements
- Python 3.x
- Windows OS
- Command Prompt (CMD) access

---

## Usage

Run the script from CMD or PowerShell:

```bash
python wifi_passwords.py
